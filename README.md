# PCNNC3
Metal-organic framework (MOF) has relevance in extensive applications such as gas adsorption, separation, and energy storage. The physically constrained neural network (PCNN) model performs excellently in the screening of MOFs. Here, we provide the code for PCNN. We have used PCNN to screen the best MOF materials for C3H6/C3H8 separation from the CoRE MOF 2019 database.
